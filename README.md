# BeToo-Apprentissage
L’objectif de ce projet est de créer une plateforme afin de faciliter l’apprentissage des  compétences des futurs étudiants de SIO. Celle-ci permet notamment de rendre les étudiants plus  autonomes face à leur formation et de faciliter l’évaluation de ces derniers par les enseignants à  travers des cours et des activités en ligne.
